Native desktop Markdown
No browser chrome. No nested window.
The current app fills the desktop surface, keeps files local, and renders source plus preview side by side.
Workspace toggle Open files when you need context, collapse them when writing.
Inspector toggle Hide document details without losing the active note.
Expand the writer or rendered page Editor, split, and preview controls make room for Markdown source or HTML output.
Rendered from plain text
Graphs, color, and underline are visible.
Mermaid graph Architecture diagrams stay in the `.md` file.
Text background `==highlight==` becomes the green note emphasis.
Underline `++observable++` renders as underlined text.
